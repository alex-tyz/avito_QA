# Автоматические E2E-тесты таск-трекера

Проект содержит E2E-тесты веб-приложения таск-трекера  
(https://avito-tech-internship-psi.vercel.app).

Покрытые сценарии:

1. Создание задачи и поиск её в списке задач (`CreateTaskTest`)
2. Переход на доску проекта из верхнего меню (`NavigateBoardTest`)
3. Открытие карточки задачи из списка (`OpenTaskTest`)
4. Поиск задачи по названию на странице «Список задач» (`SearchTaskTest`)

## Стек

- Java 17
- Maven
- Selenium WebDriver 4.25.0
- JUnit 5
- Google Chrome

## Структура

- `src/test/java/BaseTest.java`  
  Базовый класс:
  - создаёт и настраивает `WebDriver`;
  - открывает базовый URL приложения перед тестами;
  - закрывает браузер после выполнения тестов.

- `src/test/java/CreateTaskTest.java`
  - создаёт задачу с уникальным названием (8 случайных символов);
  - через поиск проверяет, что задача появилась в разделе «Все задачи».

- `src/test/java/NavigateBoardTest.java`
  - переходит на страницу «Проектные доски»;
  - открывает доску проекта и проверяет наличие колонок To Do / In Progress / Done.

- `src/test/java/OpenTaskTest.java`
  - находит задачу в списке и открывает её карточку;
  - проверяет, что карточка отобразилась.

- `src/test/java/SearchTaskTest.java`
  - выполняет поиск задачи по названию;
  - проверяет, что найдена задача с нужным текстом.

## Требования

- установлен JDK 17;
- установлен Maven;
- установлен Google Chrome;
- в `BaseTest` указан корректный базовый URL (например, `https://avito-tech-internship-psi.vercel.app`).
