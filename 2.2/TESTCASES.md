## TC-01: Создание задачи и поиск по названию

**Цель:** убедиться, что задача создаётся и её можно найти в списке задач.  
**Автоматизация:** `CreateTaskTest`.

**Предусловия:**

- Приложение доступно по базовому URL.

**Шаги:**

1. Открыть базовый URL (страница «Список задач»).
2. Нажать кнопку «Создать задачу» в шапке.
3. В поле «Название» ввести уникальную строку длиной 8 символов.
4. В поле «Описание» ввести произвольный текст.
5. С клавиатуры (TAB + стрелка вниз + Enter) выбрать значения в полях:
   - «Проект»
   - «Приоритет»
   - «Статус»
   - «Исполнитель»
6. Нажать кнопку «Создать».
7. Перейти в раздел «Все задачи».
8. В поле поиска ввести то же самое название задачи и выполнить поиск.

**Ожидаемый результат:**

- Форма создания задачи закрывается без ошибок.
- В списке задач отображается строка с указанным названием задачи.

## TC-02: Переход на доску проекта

**Цель:** проверить, что пользователь может открыть доску проекта.  
**Автоматизация:** `NavigateBoardTest`.

**Предусловия:**

- Приложение открыто (любая основная страница таск-трекера).

**Шаги:**

1. В верхнем меню нажать пункт «Проектные доски».
2. Открыть одну из досок (например, первую в списке).

**Ожидаемый результат:**

- Открывается страница доски проекта.
- На странице видны колонки To Do, In Progress, Done.

## TC-03: Открытие карточки задачи

**Цель:** проверить открытие карточки задачи из списка.  
**Автоматизация:** `OpenTaskTest`.

**Предусловия:**

- В системе есть хотя бы одна задача.
- Открыта страница «Список задач».

**Шаги:**

1. Найти в списке любую существующую задачу.
2. Нажать на её название (или на карточку задачи).

**Ожидаемый результат:**

- Открывается карточка задачи (диалоговое окно).
- В карточке видно название задачи и описание.

## TC-04: Поиск задачи по названию

**Цель:** убедиться, что поиск по названию работает корректно.  
**Автоматизация:** `SearchTaskTest`.

**Предусловия:**

- В системе есть задачи с известным фрагментом названия (например, «Тест»).
- Открыта страница «Список задач».

**Шаги:**

1. В поле поиска ввести известный фрагмент названия задачи.
2. Выполнить поиск (кнопкой или клавишей Enter).

**Ожидаемый результат:**

- В списке отображаются задачи, в названии которых есть введённый текст.
- Минимум одна задача в списке содержит этот текст в названии.
