# Автоматизированное тестирование API сервиса объявлений

В проекте реализованы автоматизированные тесты для проверки работы API микросервиса объявлений Avito.  
Тесты написаны на Java и покрывают все четыре доступные эндпоинта.

Покрываемые эндпоинты:
- POST /api/1/item — создание объявления
- GET /api/1/item/{id} — получение объявления по идентификатору
- GET /api/1/{sellerID}/item — получение всех объявлений продавца
- GET /api/1/statistic/{id} — получение статистики объявления
## Технологии
- Java 11+
- Maven
- JUnit 5
- REST Assured
- Java HttpClient
- Gson


## Требования
Для запуска тестов требуется установленный Java 11+ и Maven.

Проверка версии Java:
java -version

Проверка версии Maven:
mvn -version

## Запуск тестов
Запуск всех тестов:
mvn test

Запуск одного тестового класса:
mvn test -Dtest=SaveItemRestAssuredTest

Запуск конкретного метода:
mvn test -Dtest=SaveItemRestAssuredTest#testSaveItem

## Дополнительно
По результатам выполнения тестов отчеты находятся в каталоге:
target/surefire-reports/
